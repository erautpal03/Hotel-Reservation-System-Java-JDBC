# Hotel Reservation System (Java + JDBC)

A console-based Hotel Reservation System built using **Java**, **JDBC**, and **MySQL**.  
This project demonstrates how a Java application connects to a database and performs CRUD (Create, Read, Update, Delete) operations.

---

## Features
- Reserve a room
- View all reservations
- Update reservation details
- Delete reservations
- Get room number by reservation ID

---

## Technologies Used
- Java
- JDBC (Java Database Connectivity)
- MySQL
- IntelliJ IDEA
